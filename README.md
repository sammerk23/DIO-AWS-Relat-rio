# DIO-AWS-Relat-rio
Projeto de relatório para redução de custos de uma empresa farmacêutica fictícia usando os produtos da AWS.

# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 30/11/2025
Empresa: Abstergo Industries 
Responsável: Kelvin Sammer Gonçalves Marques

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Kelvin Sammer Gonçalves Marques. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon EC2
- Hospedagem de servidores em Nuvem
- A migração de parte dos servidores da empresa Abstergo Industries para a nuvem da AWS pode reduzir custos e melhorar a escalabilidade do sistema, possibilitando assim que a empresa possa focar seus esforços em processos mais latentes.

Etapa 2: 
- Amazon DMS
- Migração de Banco de Dados
- Utilizando o Amazon DMS, podemos fazer a migração dos bancos de dados locais para a AWS de maneira simplificada, cortando assim custos logísticos e o tempo necessário para a migração.

Etapa 3: 
- Amazon SageMaker
- Desenvolvimento e Treinamento de modelos de ML
- Ao termos os dados da Abstergo Industries reunidos na AWS, através do SageMaker poderemos desenvolver e treinar modelos avançados de Machine Learning para acelerar a busca por novos compostos farmacêuticos, assim acelerando o desenvolvimento de novos medicamentos.



## Conclusão
A implementação de ferramentas na empresa Abstergo Industries tem como esperado redução de custos e aceleração do desenvolvimento de novos medicamentos, o que aumentará a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
